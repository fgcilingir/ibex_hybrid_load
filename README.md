# Code Repository for Çilingir *et al.* (2025)

This repository provides the scripts and parameter files used for the analyses in  
**Çilingir *et al.* (2025), "Hybridization load introduced by Alpine ibex hybrid swarms."**

## Contents
- `codes/expression_analysis` – Script used for variant classification in the expression analysis  
- `codes/load_analyses` – Scripts used for running the load analyses and visualising the results  
- `codes/simulations` – Parameter file used for NemoAge runs
